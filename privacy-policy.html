Webpage JSON Extractor Privacy Policy

Last updated: 2026-05-31

This privacy policy explains how Webpage JSON Extractor handles information when you use the Chrome extension to extract structured data from webpages.

📋

Information we process

Webpage JSON Extractor processes the content of the page you choose to extract from. This can include visible text, table cells, links, images, and other structured elements needed to build the preview and export.

🛡️

How the data is used

The data is used only to provide the extraction, preview, cleanup, and export features you request. The extension does not sell your data.

🗑️

Data retention

The core workflow is designed to be local-first. Extracted content is processed in your browser for the purpose of generating your output. If future cloud features are introduced, they will be documented separately.

🔗

Third-party sharing

We do not intentionally share extracted page data with third parties for advertising or profiling. Optional future integrations, such as webhooks or automation tools, will only send data when you explicitly configure them.

Contact

If you have questions about this policy, contact the developer at the support email listed in the Chrome Web Store listing.
